Overview

除了视频播放view之外，看到的所有 view 都加在这个 view 上；这个 view 可以响应单击、双击、滑动等手势以及处理相关冲突 整体分为两层view，参见下面成员注释，其中最重要的是 controlView：part 中控制功能的控件会加到这个 view 上, 他控制着整体控件的消失和出现
