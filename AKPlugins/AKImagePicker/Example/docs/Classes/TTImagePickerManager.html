  • -getPhotoWithAsset:completion:

    根据Asset得到照片,默认本地没有，则会去icloud同步 默认尺寸为 屏幕宽度 *3 注意，所有方法里，只有默认尺寸图片做了缓存，缓存会写两次，isDegraded为YES时候会写一次，如果isDegraded为NO，高清还会写一次。

    Declaration

    Objective-C

    Swift

    Parameters

    asset

    图片或者视频对象
    completion

    isDegraded标识是否是低品质图，如果有高品质图会二次回调这个completion。

    Return Value

    请求的id,可以用这个id来cancel这次请求
